## Getting started
* Clone
* cd /code/build/ -> npm install
* cd /code/dev/ -> npm install
* cd /code/prod/ -> npm install
* cd /code/

** For developing: npm start / npm dev
** For building: npm build
** To test production: npm prod
** For tests: npm test
